#Test 1: Enviar datos a la base de datos desde la placa
##Descripción: Verificar que el módulo ESP32 pueda sensar la temperatura y enviar los datos correctamente a la base de datos.
##Objetivo: Asegurar que los datos sensados se almacenen en la base de datos sin errores y con el formato correcto.
##Pasos:
	1) Simular una lectura de temperatura en el ESP32.
	2) Enviar los datos a la base de datos.
	3) Verificar que los datos se hayan guardado correctamente en la 	base de datos.
##Criterio de éxito: Los datos enviados coinciden con los datos almacenados en la base de datos.
